# FeeModel

Specifies the pricing model used for the calculation of the final fee.

## Example Usage

```python
from moovio_sdk.models.components import FeeModel

value = FeeModel.FIXED
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `FIXED`    | fixed      |
| `BLENDED`  | blended    |
| `VARIABLE` | variable   |
