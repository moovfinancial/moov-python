# IssuingIntervalLimit

Specifies the time frame for the velocity limit. Currently supports only per-transaction limits.

## Example Usage

```python
from moovio_sdk.models.components import IssuingIntervalLimit

value = IssuingIntervalLimit.PER_TRANSACTION
```


## Values

| Name              | Value             |
| ----------------- | ----------------- |
| `PER_TRANSACTION` | per-transaction   |
