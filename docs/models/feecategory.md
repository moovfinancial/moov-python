# FeeCategory

Specifies the category in which the fee resides, whether that is `card-acquiring`, `card-push`, `card-pull`, `ach`, `rtp`, `card-other`, `network-passthrough`, or `other`


## Values

| Name                  | Value                 |
| --------------------- | --------------------- |
| `CARD_ACQUIRING`      | card-acquiring        |
| `CARD_PUSH`           | card-push             |
| `CARD_PULL`           | card-pull             |
| `ACH`                 | ach                   |
| `RTP`                 | rtp                   |
| `CARD_OTHER`          | card-other            |
| `NETWORK_PASSTHROUGH` | network-passthrough   |
| `OTHER`               | other                 |
