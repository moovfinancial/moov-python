## ping

A simple ping endpoint to check auth

### Example Usage

```python
from moov import Moov
import os

s = Moov(
    gateway_auth=os.getenv("MOOV_GATEWAY_AUTH", ""),
)

res = s.ping.ping()

if res is not None:
    # handle response
    pass

```
